# セフトリアキソンPBPKシミュレーター

セフトリアキソン（ceftriaxone, CTRX）の**治療効果指標**と**偽胆石リスク指標**を、同一の簡略化生理学的薬物動態（physiologically based pharmacokinetic, PBPK）モデル上で相対比較するための、教育用・探索用Webアプリケーションです。

本アプリは、卒業研究
**「セフトリアキソンの治療効果と偽胆石リスクを同時評価するPBPKシミュレーターの構築と仮想患者解析」**
に対応する実装です。

## 本アプリの目的

本アプリは、以下を直感的に理解するための**教育的・探索的枠組み**を提供します。

- 患者背景（体重、血清アルブミン値、GFR、食事条件）の違いが、CTRXの血漿中濃度推移にどう影響するか
- 抗菌薬としての有効性指標である **%fT>MIC** が、投与量・投与間隔・患者背景によってどう変わるか
- 胆嚢内CTRX蓄積の単純化指標として定義した **飽和指数（saturation index, SI）** が、食事回数や腎機能などでどう変わるか
- **治療効果と偽胆石リスクのトレードオフ** を、仮想患者シナリオ上でどのように考えるか

## モデルの位置づけ

本アプリは**臨床予測用ツールではありません**。以下の性格を持つモデルです。

- セフトリアキソンの既報PBPK研究を参考にした**簡略化モデル**
- 胆嚢移行・胆嚢排出・Ca-CTRX沈殿リスクを扱うための**教育用拡張モデル**
- 絶対値の正確な予測よりも、**条件間の相対比較**に重きを置いたモデル

したがって、得られた結果は
**「このモデルの仮定のもとでどう見えるか」**
として解釈してください。

## モデルの概要

### 1. PBPK本体

7コンパートメントからなる簡略化PBPKモデルを採用しています。

- 動脈血
- 静脈血
- 肺
- 肝臓
- 腎臓
- 残余組織
- 胆嚢

各組織分配は well-stirred model とし、組織-血漿間分配係数（Kp）は、PK-Sim® v12 を用いて算出した値に基づき、肺・肝臓・腎臓・残余組織で共通に **0.22** を設定しています。

### 2. 蛋白結合

CTRXの蛋白結合は、**Michaelis-Menten型1サイト飽和結合モデル**で記述しています。

- CTRXはアルブミンに対して可変・飽和性の蛋白結合を示す
- 本モデルでは、その特性を**簡略化した1サイトモデル**として表現している
- ICU患者の実測挙動を完全に再現することを目的としたものではない

### 3. 排泄

- **腎排泄**：糸球体濾過に基づき、遊離型分率を考慮して記述
- **胆汁排泄**：既報PBPK研究との整合性を優先し、**総血漿中濃度ベースの固定見かけクリアランス**として実装

このため、**低アルブミン血症による遊離型分率上昇が胆汁排泄速度へ直接反映されない**構造になっています。この点は、SI指標の解釈に影響します。

### 4. 胆嚢モデル

胆嚢に関しては、以下を単純化して表現しています。

- 空腹時：Oddi括約筋閉鎖 → 胆嚢流入あり
- 食後：CCK刺激 → Oddi括約筋開放 → 胆嚢流入なし
- 食後2時間：胆嚢排出亢進

### 5. 偽胆石リスク指標

胆嚢内CTRX濃度と胆汁中Ca²⁺濃度から、**Ca-CTRX飽和指数（SI）** を計算しています。

- SI > 10.4 を準安定限界の目安として表示
- ただし、**SIの絶対値は定量的に信頼できません**
- **同一モデル上での相対比較**に用いる指標です

## 主要な評価指標

### 有効性指標

- **%fT>MIC**：遊離型薬物濃度がMICを超えている時間の割合
- **AUCss,free**：定常状態1投与間隔における遊離型濃度のAUC

### 安全性指標

- **最大SI**
- **SI超過時間割合**
- 胆嚢内CTRX濃度推移

## 仮想患者シナリオ

卒業研究では、以下の3群の仮想患者を設定しています。

1. **体格良好・腎機能正常患者**
2. **高齢・低栄養患者**
3. **重症患者（絶食・低アルブミン血症・腎機能低下）**

詳細は `仮想患者シナリオ.md` を参照してください。

## 解釈上の重要な注意

### 1. ALBの影響

低アルブミン血症では、一般に遊離型分率（fu）は上昇します。
しかし、本モデルでは

- 胆汁排泄を固定CLで実装している
- KpをALB非依存の定数として扱っている

ため、**ALB変動の正味の影響は実臨床より単純化されています**。

特に、**SIに対するALBの影響は、生理学的に期待される方向と逆転しうる**ことに注意してください。

### 2. SIの絶対値

本モデルのSIは、胆嚢流入率・胆嚢排出速度・胆汁中Ca²⁺濃度などの仮定に依存します。

したがって、

- **絶対的な偽胆石発症確率の予測には用いない**
- **条件間の比較に限定して解釈する**

ことが必要です。

### 3. 臨床判断には使用しない

本アプリは教育・研究用途です。実際の投与設計や患者管理の意思決定に直接用いてはいけません。

## 実行方法

```bash
mamba create -n ctrx_pbpk python=3.10 -y
mamba activate ctrx_pbpk
pip install streamlit plotly scipy pandas numpy
streamlit run streamlit_app.py
```
